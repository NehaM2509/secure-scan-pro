# SecureScan Pro

A Web Vulnerability Scanner built using Python and Streamlit.

## Features
- Detects XSS vulnerabilities
- Detects SQL Injection vulnerabilities
- Checks missing security headers
- Crawl-based scanning
- Interactive dashboard
- JSON report download

## Tech Stack
- Python
- Streamlit
- BeautifulSoup
- Requests
- Plotly

## Run Locally

pip install -r requirements.txt
streamlit run secure_app.py
